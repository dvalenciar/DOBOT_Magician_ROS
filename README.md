# DOBOT Magician ROS

![Static Badge](https://img.shields.io/badge/mission-Automatizar%20el%20corte%20de%20manos%20de%20banano-purple)

![GitHub contributors](https://img.shields.io/github/contributors/dvalenciar/DOBOT_Magician_ROS)
![GitHub last commit](https://img.shields.io/github/last-commit/dvalenciar/DOBOT_Magician_ROS)

Desarrollar un sistema robótico automatizado que utilice un DOBOT Magician para cortar y separar las manos de un racimo de banano orito (racimo pequeño, adecuado al tamaño y capacidad del robot). El sistema usará una cámara para localizar cada mano mediante visión artificial, y un end effector diseñado específicamente para este proyecto, que combine capacidades de corte y sujeción para realizar el corte en el punto de unión y depositar cada mano cortada en una zona de recolección
